Context: In ServiceTitan, an Estimate is created during a service call and contains one or more options for the customer. When a customer approves a quote, the estimate status becomes Sold. For installation or project work, a separate Install Job should be created from the sold estimate. Open estimates represent opportunities that haven't been converted to revenue yet. An estimate's Opportunity Status tracks whether a follow-up was attempted (Not Attempted, Contacted, Won, Dismissed, Unreachable).

Section 4 — Sold Estimates Without an Install Job (833 estimates, $2.9M)
What This Means: When a customer accepts a quote for installation work, ServiceTitan should automatically generate an Install Job linked to the estimate. If the Install Job field is blank on a sold estimate, either (1) the job was never created, meaning the work was never scheduled or performed, or (2) the estimate was linked to a service job rather than an install job and the workflow wasn't completed. Either scenario means revenue may be falling through the cracks.
